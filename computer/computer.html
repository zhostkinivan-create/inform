Как работает компьютер?

Компьютер работает по принципу цикла выполнения инструкций:

1

Ввод (Input)

Данные поступают через устройства ввода: клавиатура, мышь, микрофон.

2

Обработка (Processing)

Процессор получает данные, выполняет вычисления и логические операции.

3

Хранение (Storage)

Данные временно хранятся в оперативной памяти во время работы.

4

Постоянное хранение

Важные данные сохраняются на жёсткий диск для долговременного хранения.

5

Вывод (Output)

Результаты выводятся на устройства вывода: монитор, принтер, колонки.

Принцип работы RAM:

  • Хранит данные во время работы компьютера
  • Прямой доступ к любой ячейке памяти
  • Быстрая скорость чтения/записи
  • Энергозависимая память (данные теряются при выключении)

Скорость доступа: ~10 ns

Работа жёсткого диска

Принцип работы HDD:

  • Магнитные пластины вращаются со скоростью 5400-7200 об/мин
  • Считывающие головки двигаются над поверхностью пластин
  • Данные записываются в виде магнитных областей
  • Механическое устройство — есть движущиеся части

Скорость вращения: 7200 RPM

Справочник по компонентам компьютера

Центральный процессор (CPU)

Назначение: Выполнение программ и обработка данных.

АЛУ

Арифметико-логическое устройство выполняет все математические и логические операции.

Устройство управления

Управляет выполнением инструкций, координирует работу всех компонентов процессора.

Регистры

Сверхбыстрая память внутри процессора для хранения промежуточных результатов.

  • R0-R7 — регистры общего назначения
  • PC — счётчик команд (адрес следующей инструкции)
  • SP — указатель стека
  • FLAGS — флаги состояния

Пример инструкций:

ADD R1, R2, R3  // Сложение: R1 = R2 + R3
SUB R1, R2, R3  // Вычитание: R1 = R2 - R3
LOAD R1, 100    // Загрузка из памяти
STORE R1, 200   // Сохранение в память
JMP label       // Переход к метке

Оперативная память (RAM)

Назначение: Временное хранение данных и программ во время работы компьютера.

Характеристики

  • Быстрая — доступ за наносекунды
  • Энергозависимая — данные теряются при выключении
  • Прямой доступ — можно обратиться к любой ячейке

Организация

Память разделена на ячейки, каждая имеет уникальный адрес:

Адрес   Данные
0x0000: [программа]
0x0100: [данные]
0x0200: [стек]

Типы памяти

  • DRAM — динамическая, требует обновления
  • SRAM — статическая, быстрее но дороже
  • DDR4/DDR5 — современные типы

Жёсткий диск (HDD)

Назначение: Постоянное хранение данных, программ и операционной системы.

Устройство

Состоит из вращающихся пластин и считывающих головок.

  • Пластины — покрыты магнитным слоем
  • Дорожки — концентрические круги
  • Секторы — участки дорожек (обычно 512 байт)

Характеристики

  • Большой объём — до нескольких терабайт
  • Медленный доступ — миллисекунды
  • Энергонезависимая — данные сохраняются
  • Механическая — содержит движущиеся части

SSD

Твердотельные накопители — современная альтернатива:

  • Быстрее — нет движущихся частей
  • Надёжнее — устойчивы к ударам
  • Дороже — за ту же ёмкость

Устройства ввода/вывода

Назначение: Взаимодействие компьютера с пользователем и внешним миром.

Устройства ввода

  • Клавиатура — ввод текста и команд
  • Мышь — управление указателем
  • Сканер — оцифровка документов
  • Микрофон — запись звука
  • Камера — захват видео

Устройства вывода

  • Монитор — вывод изображения
  • Принтер — печать на бумаге
  • Колонки — воспроизведение звука
  • Проектор — вывод на большой экран

Интерфейсы подключения

  • USB — универсальная последовательная шина
  • HDMI — цифровое видео и аудио
  • Ethernet — сетевое подключение
  • Bluetooth — беспроводное соединение
