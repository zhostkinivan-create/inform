Основы систем счисления

Двоичная система

Основание: 2, цифры: 0, 1. Используется в компьютерах для представления данных.

1010₂ = 1×2³ + 0×2² + 1×2¹ + 0×2⁰ = 8 + 0 + 2 + 0 = 10₁₀

Восьмеричная система

Основание: 8, цифры: 0-7. Удобна для представления двоичных данных.

377₈ = 3×8² + 7×8¹ + 7×8⁰ = 192 + 56 + 7 = 255₁₀

Шестнадцатеричная система

Основание: 16, цифры: 0-9, A-F. Широко используется в программировании.

FF₁₆ = 15×16¹ + 15×16⁰ = 240 + 15 = 255₁₀
