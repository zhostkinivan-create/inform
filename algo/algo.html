function bubbleSort(arr) {
    for (let i = 0; i < arr.length; i++) {
        for (let j = 0; j < arr.length - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
}

procedure bubbleSort(A: list)
    n = length(A)
    for i = 0 to n-1
        for j = 0 to n-i-2
            if A[j] > A[j+1]
                swap(A[j], A[j+1])
            end if
        end for
    end for
end procedure

Описание

Самый простой алгоритм сортировки. Последовательно сравнивает соседние элементы и меняет их местами, если они находятся в неправильном порядке. Процесс повторяется до тех пор, пока массив не будет отсортирован.

Как работает

  1. Начиная с первого элемента, сравниваем его со следующим
  2. Если текущий элемент больше следующего, меняем их местами
  3. Переходим к следующей паре элементов
  4. Повторяем процесс для всего массива
  5. После каждого прохода наибольший элемент "всплывает" в конец
  6. Повторяем для оставшейся части массива

Преимущества

  • Простота реализации
  • Не требует дополнительной памяти
  • Устойчивая сортировка
  • Хорошо работает на почти отсортированных массивах

Недостатки

  • Очень медленная для больших массивов
  • Делает много лишних сравнений и перестановок
  • Плохая производительность в худшем случае
